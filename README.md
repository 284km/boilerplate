boilerplate
===========

This is my application's boilerplate for something. It's ready to use:

- requirejs
- jquery
- underscore
- backbone
- sass
- slim
- coffee-script
- sinatra w/ reloader

Getting Start
-------------

    $ bundle
    $ bower install

Want to use other js libralies?
-------------------------------

- edit bower.json
- run "bower install"
- edit views/application.coffee w/ requirejs
